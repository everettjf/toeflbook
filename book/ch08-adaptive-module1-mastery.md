## 1 · Why Module 1 is the score lever

The 2026 adaptive engine works like this:

1. You start Module 1 (mid-difficulty items).
2. The engine measures your accuracy on Module 1.
3. If you do well, Module 2 unlocks at **higher difficulty** — your score ceiling rises.
4. If you stumble in Module 1, Module 2 stays at lower difficulty — your ceiling caps.

**Consequence:** a "silly mistake" in Module 1 hurts more than an outright wrong answer in Module 2. The engine reads Module 1 stumbles as "this person doesn't have mastery" → caps the ceiling.

This chapter rewires your practice toward **single-pass accuracy under exam conditions** — not "did I learn this concept" but "did I get it right the first time, under time, when it counted."

## 3 · Modified weekly plan (Module 1 emphasis across all skills)

**Three changes:**

1. **Every MAIN block is "Module 1 only"** — not full sections. Quality over volume.
2. **Daily 5-min "careless-error review"** between Anki and Shadowing — re-read yesterday's tagged careless errors aloud, articulate the trap.
3. **Sunday MAIN** = mini-full mock (1 R Module 1 + 1 L Module 1 + 1 Email, all back-to-back, simulating fatigue stack).

### Weekly cadence

- **Sunday** · MAIN = **Mini-full mock** (R-M1 + L-M1 + Email) · SECONDARY = Discussion
- **Monday** · MAIN = **Reading Module 1** · SECONDARY = **Writing Build-a-Sentence**
- **Tuesday** · MAIN = **Listening Module 1** · SECONDARY = Speaking — Interview
- **Wednesday** · MAIN = **Writing Discussion** · SECONDARY = Reading Module 1 (timed accuracy drill)
- **Thursday** · MAIN = **Speaking L&R + Interview** · SECONDARY = Listening Module 1
- **Friday** · MAIN = **Reading Module 1** · SECONDARY = Speaking
- **Saturday** · MAIN = L+S combined (real material) · SECONDARY = **Writing — Email**

Weekly counts: R = 3 MAIN + 1 SEC; L = 2 MAIN + 1 SEC + 1 combined; W = 2 MAIN + 2 SEC; S = 1 MAIN + 2 SEC + 1 combined. ✅

## 6 · Common pitfalls

1. **Don't speed up Module 1.** The engine doesn't care if you finish 30 seconds early. It cares about accuracy. Allocate generously, then re-check before submitting.
2. **Careless errors come in clusters.** When you misread one item, you're often in a low-attention state — the next 2–3 items are at risk too. Build a habit of taking a 5-sec micro-reset after every error you catch.
3. **Time is a tool, not a goal.** Don't say "I finished R-M1 in 14 min so I'm great." Say "I got 9/10 in 16 min so I'm great." Speed only matters if accuracy is locked.
4. **AI scoring is noisier than it looks.** Compare 3 attempts before trusting a trend. A single bad score doesn't mean regression.
5. **Don't burn out before Ch.9.** Ch.9 is full mocks ×4 — high-energy. If Ch.8 leaves you drained, you'll bomb Ch.9. Pace, hydrate, sleep.
