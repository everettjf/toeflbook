# Ch.1 · Diagnosis & System Setup (5/26 – 5/31)

> **Days:** 6 (Day 1 = Tue 5/26, Day 6 = Sun 5/31)
> **Theme:** Get tools installed, get a baseline, lock in habits — fast
> **Skill coverage:** Every day touches ≥ 2 skills (Anki + Shadowing automatically cover vocab + L+S)
> **Deliverables:** 4-skill baseline scores + weak-skill profile + 90-min daily flow operational

> ⏱️ **Why only 6 days?** This plan starts 5/26 but keeps the fixed 9/26 exam date, so the setup chapter is compressed: the baseline-and-diagnose arc is done in one tight week instead of two. Every Ch.1 deliverable is preserved — there is just no "slack week." Ch.2 onward run their full 14-day length unchanged.

## 5 · Common pitfalls in this chapter

1. **Don't read strategy tips before the baseline mock.** You want "naked score now," not "score when prepared."
2. **One Anki deck at a time.** Import 4000EEW first; add Barron's only on Day 5 once the habit holds.
3. **Shadowing week 1 feels terrible — that's normal.** Research shows real gains at 4–6 weeks. Don't quit after 3 sessions.
4. **AI speaking scores are not truth.** They're reliable on delivery/fluency, weaker on topic development. Track the trend, not the absolute number.
5. **Don't wait to "be ready" to write the error log.** Write it tonight — 3 lines beats 0 lines by infinity.
6. **The compressed setup week has no slack.** If you must skip a block, skip the 20-min real-material block — never the baseline mocks (Days 2–3) or the weak-skill profile (Day 4).
